# Limitations Language Library

## Purpose

This library provides reusable status and limitation wording for public-safe civic AI artifacts.

## Status Language

| Status | Safe wording |
| --- | --- |
| Planned | `Status: Planned / Not released. This item is a future candidate and does not represent a live model, dataset, Space, report, service, deployment, or public release.` |
| Scaffolded | `Status: Scaffolded / Not released. This repository contains structure, templates, and public-safe placeholders only.` |
| Released | `Status: Released. The public artifact exists at the linked URL and release status has been reviewed.` |
| Paused | `Status: Paused. Work is not active and no current release or service claim is made.` |
| Deprecated | `Status: Deprecated. The artifact remains available for historical reference and should not be used as current guidance.` |
| Private/not-public | `Status: Private / Not public. Details are intentionally excluded from public materials.` |

## Limitation Patterns

| Use case | Safe wording |
| --- | --- |
| Model card | `This card does not establish that a model is suitable for legal, medical, eligibility, enforcement, surveillance, or high-stakes decisions.` |
| Dataset card | `This card does not publish raw data, private records, or unapproved sanitized samples.` |
| Space demo | `This scaffold is not a live civic service and must not be used for sensitive personal data.` |
| Public report | `This report template is not a real Foundation report unless a reviewed report artifact is linked.` |
| Portfolio link | `Portfolio references must describe public artifacts only and must not imply client results or active services without review.` |

## Avoid

- `production-ready`;
- `deployed in schools`;
- `released model`;
- `public dataset`;
- `live assistant`;
- `verified impact`;
- `client-proven`;
- `NEURONA deployment`;
- any equivalent wording without public evidence and review.
