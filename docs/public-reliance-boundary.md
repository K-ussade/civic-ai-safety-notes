# Public Reliance Boundary

## Purpose

This document defines what the public may and may not rely on when using Foundation civic AI artifacts.

## Default Reliance Position

Unless a reviewed release states otherwise, Foundation civic AI artifacts are informational or demonstrational. They are not official determinations, legal advice, emergency systems, medical advice, school placement tools, enforcement tools, or benefits eligibility systems.

## Unsupported Reliance

| Use | Default Position |
| --- | --- |
| Emergency response | Unsupported. Use official emergency channels. |
| Legal advice | Unsupported. Consult qualified legal resources. |
| Medical advice | Unsupported. Consult qualified medical resources. |
| Benefits eligibility | Unsupported. Use official program rules and human review. |
| Education placement | Unsupported. Use school-authorized processes and human review. |
| Enforcement or surveillance | Unsupported. |
| Individualized decisions using private data | Unsupported and forbidden for public artifacts. |
| Official government action | Unsupported unless an official public authority explicitly controls the system. |

## Required Escalation Language

Safety notes should direct users to:

- official public sources;
- qualified professionals where appropriate;
- Foundation contact or issue channels for public documentation corrections;
- emergency channels for urgent safety needs.

## Public Input Rule

Public demos and examples must instruct users not to enter donor private data, student data, school private data, volunteer private data, customer data, private Foundation operations, private telemetry, secrets, or sensitive operational details.
