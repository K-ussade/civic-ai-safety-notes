# Limitation Language

## Status Block

```text
Status: [Planned / Not released | Scaffolded / Not released | Released | Paused | Deprecated | Private / Not public]
This artifact [does/does not] represent a public release.
```

## Model Limitation

```text
This model card does not establish suitability for legal, medical, eligibility, enforcement, surveillance, or other high-stakes decisions.
```

## Dataset Limitation

```text
This dataset card does not publish raw data, private records, private telemetry, donor data, student data, school private data, volunteer data, customer data, or unapproved sanitized samples.
```

## Space Limitation

```text
This Space scaffold is not a live civic service and does not process real donor, student, school, volunteer, customer, operational, or private telemetry data.
```

## Portfolio Limitation

```text
This portfolio reference describes public-safe documentation structure only. It does not claim active services, client outcomes, model releases, dataset releases, or live deployments without separate review.
```
