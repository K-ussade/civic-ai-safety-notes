# Security Policy

## Reporting Sensitive Issues

Do not open a public issue containing secrets, private records, donor data, student data, school private data, private incident information, private telemetry, infrastructure details, sealed YOSO-YAi LLC IP, private model or dataset material, or security-sensitive NEURONA operational details.

Use a private maintainer contact path until a formal Foundation security contact is published.

## Public Issue Scope

Public issues may discuss:

- wording clarity;
- checklist completeness;
- broken links;
- graph rendering;
- validation checks;
- public boundary improvements.

Public issues must not include:

- credentials, tokens, or API keys;
- private data;
- private governance drafts;
- private incident records;
- model weights;
- dataset files;
- private training corpora;
- private evaluations;
- private telemetry;
- production prompts;
- security-sensitive NEURONA operational details.

## Supported Scope

This repository is documentation-only. It does not collect data, run systems, host models, publish datasets, operate Spaces, or provide public services.
